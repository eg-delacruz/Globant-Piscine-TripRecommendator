# Globant-Piscine-TripRecommendator

Globant piscine project intended to develop a trip recomendator app using React, Typescript and TailwindCSS

## How to use

1. Run `make start` to start the development server.
2. Open your browser and navigate to `http://localhost:5173` to view the app..
3. Stop the development server: By this time, your terminal will be inside the dev container. To stop the server, press `q + Enter`. Yo should be now outside the container. If not, type `exit`.
